# Support Services Layer

This layer provides cross-cutting concerns and infrastructure services.

## Components:
- **Logging**: Serilog/NLog configuration
- **DependencyInjection**: Service registration and configuration
- **Configuration**: JSON config and user settings management
- **BackgroundTasks**: Background queue and task scheduling
- **Validation**: Input validation services
- **Utilities**: Helper classes and extensions

## Responsibilities:
- Application-wide logging
- Dependency injection container setup
- Configuration management
- Background processing
- Common utilities and helpers
