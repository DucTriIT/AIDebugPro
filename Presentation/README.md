# Presentation Layer (UI)

This layer contains all UI-related components for Windows Forms.

## Components:
- **Forms**: Main windows and dialogs
- **UserControls**: Reusable UI components (WebView Panel, AI Assistant Panel, Logs Dashboard)
- **ViewModels**: MVVM-style view models for data binding and UI logic
- **Commands**: UI command implementations

## Responsibilities:
- Main Window (Shell)
- WebView Panel (browser display)
- AI Assistant Panel (chat, insights, reports)
- Logs & Metrics Dashboard
- Command Toolbar (capture, export, re-analyze)
