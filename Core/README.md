# Core Layer

This layer contains shared models, interfaces, and common utilities used across all layers.

## Components:
- **Models**: Shared data models and DTOs
  - TelemetryModels
  - SessionModels
  - AIModels
- **Interfaces**: Core abstractions
- **Enums**: Shared enumerations
- **Constants**: Application constants
- **Exceptions**: Custom exception types

## Responsibilities:
- Define common data structures
- Provide shared abstractions
- Centralize constants and enums
- Enable loose coupling between layers
